## Keyboard Shortcut Conflicts

### Shortcuts That Work Everywhere
- **Space**, **F**, **R**, **?**, **Esc** - Work in all contexts

### Shortcuts That Don't Work in Input Fields
- All shortcuts are disabled when typing in input fields
- Click outside input field to re-enable shortcuts

### Browser Shortcuts to Avoid
- **Ctrl+F** / **Cmd+F** - Browser find (conflicts with F key)
  - Solution: Use F key without Ctrl/Cmd modifier
- **F11** - Browser full-screen (different from presentation mode)
  - Solution: Use F key for presentation mode instead
- **Ctrl+R** / **Cmd+R** - Browser refresh (conflicts with R key)
  - Solution: Use R key without Ctrl/Cmd modifier

---

## Troubleshooting Shortcuts

### Shortcut Not Working?

**Check:**
1. Is focus on the demo window? (Click on page)
2. Are you typing in an input field? (Click outside)
3. Is browser blocking the shortcut? (Try different browser)
4. Is keyboard layout correct? (US layout recommended)

**Solutions:**
- Click on demo page to ensure focus
- Close any open dialogs or overlays
- Try using mouse/trackpad as backup
- Restart browser if shortcuts stop working

### Alternative Input Methods

If keyboard shortcuts fail:
- **Mouse/Trackpad** - Click buttons in UI
- **Touch** - Tap buttons on touchscreen
- **Presentation Remote** - Use arrow keys on remote
